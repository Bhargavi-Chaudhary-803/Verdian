# Verdian, A Smart Waste Management Platform

AI-powered waste segregation and recycling optimization system.

## Features

- **Landing Page** — Hero, stats, features, CTA
- **Auth** — Citizen / Admin login with role switching
- **User Dashboard** — Green points, CO₂ tracking, activity feed, segregation guide
- **AI Scanner** — Camera, upload, or manual input → classification + disposal guidance
- **Waste Map** — Real-time hotspot grid map (Delhi zones)
- **Hotspots** — Priority zone listing with volume + trends
- **Add Waste** — Manual waste logging with point rewards
- **Analytics** — Weekly charts, category breakdown, KPIs
- **Admin Dashboard** — Control centre, critical zones, schedule, leaderboard

## Demo

Just click **Sign In** on the auth page — no credentials needed.  
Switch between **Citizen** and **Admin** roles to see both dashboards.
